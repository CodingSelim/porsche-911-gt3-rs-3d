# Porsche 911 GT3 RS — 3D Interactive Experience

An interactive 3D marketing experience built using React Three Fiber, Three.js, and Next.js to bring a static model of the Porsche 911 GT3 RS (Type 992) to life. Powered by scroll-driven kinematics, the site showcases key automotive engineering features including active aerodynamics, suspension physics, and rotational center-lock wheel speeds.
